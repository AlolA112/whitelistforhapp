This file is needed exclusively for VPN clients with routing rules. 
It is configured so that applications on the Russian Federation whitelists are routed directly, while all others are routed through a proxy.
You need to add geosite:mydirect to the direct file, and add geosite:myproxy to the proxy file. 
This list will be expanded as needed, and the main domains and the collector itself were taken from Loyalsoldier. 
Here is the path to its directory: https://github.com/Loyalsoldier/v2ray-rules-dat
The file is created in the Go programming language.
